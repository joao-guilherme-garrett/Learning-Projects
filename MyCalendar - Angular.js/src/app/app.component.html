Coding Challenge

Assignment : The goal of this exercise is to create a demo calendar application using React, Vue or Angular.

My Approach : moment.js was my library of choise for Date managing and no UI library was used to manage the precise behaviour of each component.

Forecast is only available for reminders no loger than 5 days apart from the current because of FREE user limitation for OpenWeather.

Reminders are stored on the browser local storage in separate lists by month.
